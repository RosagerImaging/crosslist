# BMad Agent Roster for the Crosslist Project

This document outlines the suggested AI agents to be created for the Crosslist project, as brainstormed with the BMad Builder.

## Debugging and Development Process Agents

This team of agents is focused on the development, maintenance, and quality assurance of the Crosslist application.

- **The Debugging Orchestrator:** Manages the entire debugging workflow, from problem identification to final validation.
- **The Inspector (Code Reviewer Agent):** Analyzes code for errors, style issues, and potential bugs.
- **The TypeScript Sage (TypeScript Expert Agent):** A specialist for resolving TypeScript-related issues.
- **The Python Paladin (Python Expert Agent):** An expert in all things Python, focused on the FastAPI backend.
- **The Strategist (Test Planner Agent):** Devises comprehensive test plans for new code changes.
- **The Builder (Test Implementer Agent):** Writes the test code based on the Strategist's plans.
- **The Guardian (Validator Agent):** Runs tests, analyzes results, and validates fixes.
- **The Monorepo Medic**: An expert in Turborepo and npm workspaces, to diagnose and fix build issues, dependency conflicts, and monorepo configuration problems.
- **The Frontend Fixer**: A specialist in Next.js, React, and Tailwind CSS, to tackle UI bugs, routing problems, and styling issues.
- **The Auth Guardian**: An expert in NextAuth.js and web security, to debug authentication flows and patch security vulnerabilities.
- **The Database Detective**: A specialist in PostgreSQL and SQLAlchemy, to investigate database-related issues, optimize queries, and manage migrations.
- **The AI Whisperer**: An expert in LangChain and prompt engineering, to debug the behavior of the AI agents and fine-tune their performance.
- **The Extension Examiner**: A specialist in Chrome extension development, to debug content scripts, background scripts, and communication with the main application.
- **The Pipeline Protector**: An expert in CI/CD and deployment pipelines, to diagnose and fix build and deployment failures.

## Feature-Building Agents

This team of agents is responsible for building the core AI-powered features of the Crosslist application.

- **The Feature Building Orchestrator:** Manages the entire feature building workflow, from idea to implementation.
- **The Crosslisting Champion**: Builds the system that takes a single listing and multiplies it across all marketplaces.
- **The Listing Optimizer Guild**: A team of agents to build the "Listing Optimizer" specialists:
  - **The Photo Phantom**: Creates the agent for optimizing product images.
  - **The Wordsmith**: Creates the agent for crafting compelling product descriptions.
  - **The Price Oracle**: Creates the agent for determining optimal pricing.
- **The Sourcing Sentinel**: Builds the system that finds profitable products to sell.
- **The Promotion Paladin (Poshmark Specialist)**: Builds the "Poshmark Bot" for social promotion.
- **The Trend Tracker**: Builds the system that identifies trending products.
- **The Liquidation Legionnaire**: Builds the system that manages and sells stale inventory.
- **The Customer Service Sentinel**: Builds the customer service chatbot.

## Implementation Phase Agents

This team of agents is focused on the architectural and structural development of the Crosslist project.

- **The Implementation Orchestrator:** Manages the entire implementation workflow, from scaffolding to deployment.
- **The Scaffolding Smith**: Handles the initial setup of new applications and packages.
- **The Database Dragon**: Designs and implements the database schema.
- **The UI Unicorn**: Crafts the user interface for the central hub.
- **The Chrome Extension Crafter**: Forges the Chrome extension for marketplace integration.
